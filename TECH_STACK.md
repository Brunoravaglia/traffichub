# Tech Stack

## Frontend
- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Data/Auth/Infra
- Supabase
- Vercel deployment

## Practical Constraints
- Keep bundle impact low on public pages.
- Prefer shared token-based styles over heavy one-off CSS.
- Use existing UI primitives (`Button`, `Tooltip`, etc.) for consistency.
